# ==============================================================================
# Readline Support (For Arrow Keys & Command History)
# ==============================================================================

# macOS: The built-in libedit often has issues with arrow keys.
# Install gnureadline for proper GNU readline support:
# pip install gnureadline

# Windows: pyreadline3 provides readline functionality:
# pip install pyreadline3

# Linux: readline is usually built-in and works correctly.

# Uncomment to enable:
